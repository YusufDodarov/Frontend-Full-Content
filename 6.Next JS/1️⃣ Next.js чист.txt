1️⃣ Next.js чист?
Next.js — ин як фреймворки JavaScript аст, ки барои сохтани вебсайтҳо ва вебаппликейшнҳо бо React истифода мешавад.
Ба ибораи дигар: React як китобхона аст, ки барои сохтани интерфейс (чӣ гуна чизҳо дар браузер намоиш дода мешаванд) истифода мешавад. Next.js он React-ро «кушода» мекунад ва бисёр чизҳоро барои сохтани сайт омода мекунад.
2️⃣ Афзалиятҳои асосии Next.js
Server-side rendering (SSR)
Меъёран, браузер пеш аз он ки сайтро нишон диҳад, сервер HTML-ро месозад.
Ин барои SEO ва суръати боркунии аввал хеле хуб аст.
Static site generation (SSG)
Сафаришгари HTML-ро пешакӣ месозад ва сервер ба корбар ҳамонро медиҳад.
Сайт хеле зуд бор мешавад.
API Routes
Шумо метавонед функсияҳои сервериро дар ҳамон лоиҳа нависед.
Масалан, барои гирифтани маълумот аз базаи додаҳо.
Routing осон
Ба ҷои насб кардани React Router, шумо танҳо фаилҳоро дар папкаи pages мегузоред ва онҳо URL-и худро мегиранд.
Оптимизацияи тасвир ва код
Next.js ба таври худкор тасвирҳоро оптимизатсия мекунад, ки сайт тез ва сабук шавад.
3️⃣ Чӣ тавр кор мекунад?
Дар React шумо ҳама чиро дар браузер месозед.
Дар Next.js шумо метавонед ба сервер ва браузер ҳамон кодро нависед, ки вебсайт ҳам зуд бошад ва ҳам SEO хуб дошта бошад.
