🖥️ Server Side Rendering (SSR)

Server Side Rendering (SSR) — ин усули рендеринг аст, ки ҳамаи намоиши маълумот 
дар сервер сурат мегирад, на дар браузер.
Яъне сервер HTML-и тайёршударо месозад ва онро пурра ба браузери корбар мефиристад.
Браузер танҳо онро нишон медиҳад, дигар кор намекунад.

⚙️ Чӣ тавр SSR кор мекунад:
1️⃣ Корбар (браузер) ба сервер дархост мефиристад — масалан, https://example.com/users.
2️⃣ Сервер кодро иҷро мекунад (масалан, React бо Next.js ё Express бо шаблонҳо).
3️⃣ Сервер HTML-и тайёр ва пурра пуршуда месозад:
<html>
  <body>
    <h1>Users List</h1>
    <ul>
      <li>Ali</li>
      <li>Ahmad</li>
    </ul>
  </body>
</html>
4️⃣ Ин HTML тайёр ба браузер фиристода мешавад.
5️⃣ Браузер онро фавран нишон медиҳад (ҳатто бе JavaScript).
