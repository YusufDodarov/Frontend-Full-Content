useCallback чист?

useCallback — ин як hook-и React аст, ки функсияро мемоизатсия мекунад, яъне ҳамон як 
версияи функсияро нигоҳ медорад то даме ки dependency-ҳо иваз нашаванд.
Масъалае, ки useCallback ҳал мекунад
Дар React, ҳар render → функция аз нав сохта мешавад.
Ин метавонад сабаб шавад:
re-render-и нолозими child-component
вайрон шудани React.memo ва агар запрос равон карда бошем беҳуда он боз кор мекунад

Синтаксис 
const memoFn = useCallback(() => {
  // code
}, [deps])

Мисоли одди бе useCallback
import React, { useState } from 'react';

export default function Counter() {
  const [count, setCount] = useState(0);
  console.log("Component render");

  return (
    <div>
      <p>Count: {count}</p>
      <button onClick={()=> setCount(count + 1)}>Increment</button>
    </div>
  );
}
