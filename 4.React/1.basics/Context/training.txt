бидуни props паҳн кунем. Дар асос барои мудирияти стейтҳо истифода мешавад ки стейтҳои 
глобали созем барои мисол:
1.user (логиншуда ё не)
2.theme (dark / light)
3.language
4.settings 

exsample Бе истифодаи UseContext:
App.jsx
import React, { useState } from "react";
import ComponentsA from "./Components/ComponentsA";
function App() {
  const [state, SetState] = useState("Yusuf");
  return (
    <div className="p-1"><ComponentsA states={state} setStates={SetState} /></div>
  );
}
export default App;

ComponentsA.jsx
